# Kano-Modell

## Kundenzufriedenheit

## :dart: ​Ziel

**Produktmerkmale nach ihrem Einfluss auf die Kundenzufriedenheit zu kategorisieren** 

## Beschreibung

Das Modell besagt, dass es bei einem Produkt bzw. bei einer Diensleistung fünf verschiedene Merkmale gibt, die unterschiedliche Auswirkungen auf die Zufriendenheit des Kunden haben. Diese sind:

+ Basismerkamle
+ Leistungsmerkmale
+ begeisterungsmerkmale
+ Unerhebliche Merkmale
+ Rückweisungsmerkmale
